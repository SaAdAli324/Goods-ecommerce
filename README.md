# 🛒 ProCart: Full-Stack MERN E-commerce System

A comprehensive, full-stack e-commerce solution built with the MERN stack. This project features a high-performance customer storefront, a secure backend, and a feature-rich Admin Dashboard for business management.

## 🚀 Key Features

### 💻 Customer Storefront
* **Complete CRUD Functionality:** Browse products, view details, and manage cart items.
* **Smart Shopping Cart:** Persistent cart state with real-time price calculations.
* **Product Search & Filtering:** Find products by category or name.
* **User Authentication:** Secure signup and login using JWT (JSON Web Tokens).

### 📊 Admin Dashboard
* **Order Analytics:** Visualized stats for total orders, revenue, and pending deliveries.
* **Product Management:** Full control to add, edit, or delete products from the inventory.
* **Order Tracking:** View all customer orders and update their fulfillment status.
* **Customer Insights:** Overview of registered users and their activity.

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS (Responsive Design)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose ODM)
* **Authentication:** JWT & Bcrypt (Password Hashing)
* **State Management:** React Context API / Redux
